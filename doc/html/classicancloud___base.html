Detailed Description

Macro to perform output loggin.

Base class that contains common functionality for icancloud classes. This functionality includes:

  • Manage the path between modules for the received/sent messages.
  • Send response/request messages to the corresponding destination module.
  • Recognize response/request messages.

void icancloud_Base::handleMessage ( cMessage *  msg)
protectedvirtual

This method classifies an incoming message and invokes the corresponding method to process it.

For self-messages, it invokes processSelfMessage(sm); For request-messages, it invokes processRequestMessage(sm); For response-messages, it invokes processResponseMessage(sm);
